    PLUS (C code)¶

    Kneron Platform Library United Software

    1. Description¶

    PLUS is a API-based software library, developed by Kneron, which allows users to manipulate the AI devices.

    2. Install Dependency¶

    Before execute any examples, please install the dependent module and wheel file first.

    1. Install dependencies
    $ sudo pacman -S python-pip python-numpy python-opencv
    1. Install wheel
    $ sh install_package.sh

    3. Privilege Setting¶

    Before running any examples, please follow the instruction below to eliminate the privilege issue.

    1. Add following rules in /etc/udev/rules.d/10-local.rules manually

      SUBSYSTEM=="usb",ATTRS{product}=="Kneron KL520",ATTRS{idVendor}=="3231",ATTRS{idProduct}=="0100",MODE="0666"
SUBSYSTEM=="usb",ATTRS{product}=="Kneron KL720l",ATTRS{idVendor}=="3231",ATTRS{idProduct}=="0200",MODE="0666"
SUBSYSTEM=="usb",ATTRS{product}=="Kneron KL720",ATTRS{idVendor}=="3231",ATTRS{idProduct}=="0720",MODE="0666"
SUBSYSTEM=="usb",ATTRS{product}=="Kneron KL630",ATTRS{idVendor}=="3231",ATTRS{idProduct}=="0630",MODE="0666"
SUBSYSTEM=="usb",ATTRS{product}=="Kneron KL730",ATTRS{idVendor}=="3231",ATTRS{idProduct}=="0732",MODE="0666"

    2. Apply the new rules by following commands (Or reboot the host)

      $ sudo udevadm control --reload-rules
$ sudo udevadm trigger
